<h1>Site prestador de serviços</h1>
<br>
<p>  Criei este site com HTML e CSS para apresentar um modelo de site de prestação de serviços.</p>
<p> É um site responsivo(adaptado para tablets e celulares), com algumas funções de endereçar para um whatsap, acessar outra aba em "Nossos serviços" e também implementei uma função com JavaScript que ao clicar no "Fale conosco" ou "Entre em contato" o site traz para a tela um formulário para preencher que envia diretamente para o e-mail.</p>
